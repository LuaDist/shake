Introduction

Shake is a simple Lua test engine that assumes that tests only use standard assert and print calls. If you are looking for a xUnit style framework, check lunit and luaunit instead.

Installation

The Shake source distribution contains two components. The first one is a Lua module and command line runner, the second is a Kepler based runner.
